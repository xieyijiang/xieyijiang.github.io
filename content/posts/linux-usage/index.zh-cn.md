## 部署 Git

> <cite>参考：如何在 Debian 11 上安装和配置 Git[^1]</cite>

[^1]: [如何在 Debian 11 上安装和配置 Git](https://cn.linux-console.net/?p=13664)

### 方法一：使用 apt 安装

更新升级系统

```shell
sudo apt update && sudo apt upgrade
```

安装 Git

```shell
sudo apt install git
```
